# 定西美年人员打卡对比工具

这是一个用于对比定西美年公司人员打卡情况的工具，可以帮助您快速识别未打卡人员。

## 功能特点

- ✅ 上传并解析 Excel 文件
- ✅ 按公司、部门和日期筛选数据
- ✅ 一键遍历填入功能
- ✅ 对比人员名单，筛选出未打卡人员
- ✅ 人员名单的增删改补管理
- ✅ 导出 Excel 和 SVG 格式的统计数据

## 使用说明

### 基本功能使用
1. 打开网页后，点击 "上传并解析" 按钮上传包含打卡数据的 Excel 文件
2. 在筛选条件中选择 "定西" 公司（支持各种格式的定西公司名称）和具体日期
3. 点击 "对比人员名单" 按钮查看未打卡人员

### 人员名单管理
1. 点击 "管理人员名单" 按钮
2. 在 "添加人员" 标签页中输入姓名并点击 "添加" 按钮
3. 在 "删除人员" 标签页中输入姓名并点击 "删除" 按钮
4. 在 "更新人员" 标签页中输入旧姓名和新姓名并点击 "更新" 按钮
5. 在 "查看名单" 标签页中查看当前所有人员名单

## 技术说明

- 使用纯 HTML、CSS 和 JavaScript 开发
- 使用 SheetJS 库解析 Excel 文件
- 使用 Bootstrap 进行样式设计
- 使用 SortableJS 实现表格行拖拽排序

## 部署说明

### 部署到 GitHub Pages
1. 将项目克隆到本地
2. 初始化 git 仓库并推送到 GitHub
3. 在 GitHub 仓库设置中启用 Pages 功能
4. 选择 "main" 分支作为源分支
5. 等待 GitHub 自动部署完成

### 本地运行
1. 进入项目目录
2. 启动本地服务器，例如：
   ```bash
   python3 -m http.server 8000
   ```
3. 在浏览器中访问 http://localhost:8000

## 注意事项

- 确保 Excel 文件中包含姓名列（列名可以包含 "姓名"、"名字" 或 "人员" 等关键词）
- 确保选择了正确的日期范围
- 支持的 Excel 格式：.xlsx, .xls
